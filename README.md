# ENCS3320 – Network Programming Project

## Overview
This project was developed for the ENCS3320 Computer Networks course using Python and Socket Programming.

## Features
- Network analysis using Wireshark
- TCP-based HTTP Web Server
- HTML and CSS web pages
- HTTP GET request handling
- 200 OK, 404 Not Found, and 307 Redirect responses
- TCP/UDP Multiplayer Guessing Game

## Technologies Used
- Python
- Socket Programming
- TCP/IP
- UDP
- HTTP Protocol
- HTML
- CSS
- Wireshark
